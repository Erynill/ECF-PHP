# ECF Bibliothèque
## INSTALLATION

Clone le git puis "composer install" pour installer les librairies PHP

Créer un .env, ajouter les variables suivants et changer leurs valeurs pour correspondre à votre projet : 
- Add ":" after the type
- DB_TYPE="yourdb:"
- Adress + port
- DB_HOST="host:port"
- DB Name
- DB_NAME="dbname"
- Logins to connect in db
- DB_USER="user"
- DB_PASS="pass"
